## Usage

> run

``` sh
bash <(curl -L https://raw.githubusercontent.com/samwhelp/ubuntu-gnome-shell-adjustment/refs/heads/main/demo/scripts/layout/gnome-shell-layout-maccity/gnome-shell-layout-maccity.sh)
```

> or run

``` sh
bash <(wget -q https://raw.githubusercontent.com/samwhelp/ubuntu-gnome-shell-adjustment/refs/heads/main/demo/scripts/layout/gnome-shell-layout-maccity/gnome-shell-layout-maccity.sh -O - )
```


> or run

``` sh
bash -c "$(curl -fsSL https://raw.githubusercontent.com/samwhelp/ubuntu-gnome-shell-adjustment/refs/heads/main/demo/scripts/layout/gnome-shell-layout-maccity/gnome-shell-layout-maccity.sh || wget -q https://raw.githubusercontent.com/samwhelp/ubuntu-gnome-shell-adjustment/refs/heads/main/demo/scripts/layout/gnome-shell-layout-maccity/gnome-shell-layout-maccity.sh -O - )"
```


> or run

``` sh
bash -c "$(wget -q https://raw.githubusercontent.com/samwhelp/ubuntu-gnome-shell-adjustment/refs/heads/main/demo/scripts/layout/gnome-shell-layout-maccity/gnome-shell-layout-maccity.sh -O - || curl -fsSL https://raw.githubusercontent.com/samwhelp/ubuntu-gnome-shell-adjustment/refs/heads/main/demo/scripts/layout/gnome-shell-layout-maccity/gnome-shell-layout-maccity.sh)"
```
